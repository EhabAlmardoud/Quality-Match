# Quality-Match
Quality Match Coding Task

To download and use, simply run the following commands on your machine:

```bash
git clone https://github.com/EhabAlmardoud/Quality-Match.git
cd Quality-Match
npm install
```

To start the project run:
```bash
npm start
```
